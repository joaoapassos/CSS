Especificidade CSS

Regra 1

O estilo possui uma prioridade que segue primeiro vindo dentro da TAG, depois um estilo dentro da pagina HTML, e aí vai para o link do CSS.

Regra 2

Seletores mais específicos tem maior precedência sobre os menos especificos

Ids recebem mais prioridade que classes e classes mais prioriades do que p, div, h2(chamados de seletores simples) chamados no css

Porem quando o elemento é citado antes da classe ele acaba sendo muito mais importante que a class sozinha como no exemplo "p .exemplo" ao contrario de ".exemplo p" que nesse caso recebe menos prioridade que a anterior

Regra 3

Se caso use o mesmo seletor varias vezes com valores diferente o valor do ultimo sera usado

Resumo

Em ordem Id -- seletor . class -- class -- seletor simples ||||| tag -- estilo dentro do html -- link do arquivo css
